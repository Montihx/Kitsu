# Roadmap — развитие проекта KITSU

> ⚠️ **СТРАТЕГИЧЕСКИЙ ДОКУМЕНТ**
> Этот файл определяет **долгосрочное развитие проекта KITSU**.
> Используется как ориентир для:
>
> * владельца продукта
> * backend / frontend
> * **AI Copilot / Agent**
>
> ❌ Запрещено реализовывать функционал вне roadmap без обновления этого файла

---

## 1. Назначение Roadmap

Roadmap:

* фиксирует этапы развития
* предотвращает хаотичное добавление фич
* задаёт бизнес-направление
* объясняет AI-агенту, **что допустимо, а что нет**

Этот документ **НЕ маркетинг**, а план исполнения.

---

## 2. Этап 0 — Основа (Foundation)

**Статус:** обязателен перед любым релизом

### Цели:

* стабильная архитектура
* безопасный backend
* корректный плеер

### Входит:

* Next.js frontend
* FastAPI backend
* Kodik parser
* Shikimori parser
* Player с HLS.js
* базовое кеширование

❌ Запрещено на этом этапе:

* авторизация
* социальные функции
* монетизация

---

## 3. Этап 1 — MVP+ (Качество)

### Цели:

* UX без раздражающих багов
* стабильность

### Функционал:

* история просмотров
* resume playback
* избранное
* улучшенные ошибки плеера
* оптимизация запросов

---

## 4. Этап 2 — Пользователи

### Цели:

* персонализация

### Функционал:

* аккаунты
* авторизация (JWT)
* синхронизация прогресса
* профили

---

## 5. Этап 3 — Масштабирование

### Цели:

* рост нагрузки
* стабильность

### Функционал:

* горизонтальное масштабирование backend
* очереди
* advanced caching
* provider health-checks

---

## 6. Этап 4 — Монетизация (опционально)

### Возможные варианты:

* подписка
* донаты
* premium UX-фичи

❌ Запрещено:

* продажа контента
* платный доступ к видео

---

## 7. Этап 5 — Экосистема

### Функционал:

* мобильные приложения
* публичное API (read-only)
* партнёрства

---

## 8. Технический долг

* устраняется на каждом этапе
* не переносится бесконечно

---

## 9. Правила для AI-агента

* не перескакивать этапы
* не добавлять фичи из будущих этапов
* при сомнениях — откладывать

---

## 10. Статус документа

* обязателен
* стратегический
* изменяется осознанно
